
Pour exécuter l'exemple du Loader d'Oracle, il faut :

1°) copier ce répertoire sur Z:\
1°) Créer la table Aliments en exécutant la requête SQL du fichier ScriptSondageCreate
2°) dans le fichier LoaderSource, changer le compte et le mot de passe oracle
3°) exécuter la ligne du fichier LoaderSource sous l'invite de commande DOS

Le résultat est :

Les données contenues dans le fichier "Sondage.dat" sont insérées dans la table "Sondage"
